## Brute Force
- 문자열 패턴 매칭 알고리즘
- 검색 대상이 되는 문자열의 처음부터 끝까지 차례로 순회하여 일일이 비교하는 알고리즘

### 시간 복잡도
- O(nm)
  - n : 검색 대상의 문자열 길이
  - m : 총 문자열 길이
- 순회하며 모두 검색하기 때문에 정답률 굳
- 다 찾기 때문에 시간적으로 비효율적

### 관련 파이썬 코드
- 그냥.. 반복문 써서 순회하기
- 반복문을 돌다가 정답이 나오면 return
